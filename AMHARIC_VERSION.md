# Amharic Translation — Challenge Registration & Submission Flow

## Overview

When a user clicks "Register Now", they first choose a language:
- 🇬🇧 English
- 🇪🇹 አማርኛ

If they choose Amharic, all DM interactions (registration + submission + DM notifications) are in Amharic. Channel posts remain English-only.

## Implementation Notes

- Store language preference in session: `session.data.lang = 'am' | 'en'`
- Optionally persist in DB: `ALTER TABLE trading_registrations ADD COLUMN lang VARCHAR(2) DEFAULT 'en'`
- Create `src/i18n/am.ts` and `src/i18n/en.ts` with all message templates
- Use a helper: `t(session.data.lang, 'registration_complete', { nick, email, ... })`
- Channel posts (announcements, countdowns, end messages) → English only, unchanged
- Only private DM interactions use the selected language
